# House_Price_Prediction
A simple Streamlit-based web app that predicts price of your dream home using 13 features  'area', 'bedrooms', 'bathrooms', 'stories', 'mainroad', 'guestroom', 'basement', 'hotwaterheating', 'airconditioning', 'parking', 'prefarea' and 'furnishingstatus'('furnished', 'semi-furnished' and  'unfurnished').
